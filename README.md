# Employee Management Dashboard

## Overview

A React.js based Employee Management Dashboard with authentication, CRUD operations,
image upload, filtering, and printing.

## Tech Stack

- React.js
- JavaScript
- LocalStorage
- HTML5 / CSS3

## Features

- Mock Login Authentication
- Add / Edit / Delete Employees
- Image Upload with Preview
- Search & Filter (Gender, Status)
- Print Employee List
- Responsive UI

## Run Locally

1. npm install
2. npm start
3. Open http://localhost:3000

## Assumptions

- Authentication is mocked
- Data is stored in browser localStorage
